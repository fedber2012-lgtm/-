Это версия на PHP + SQLite.

Что реализовано
- общий банк тем для всех пользователей
- лобби с кодом
- подключение игроков с телефона
- кнопка ответа на телефонах
- ведущий на компьютере
- общий серверный банк тем
- удаление только своих тем, для админа — любых
- админка с банами
- бан на дни, навсегда, досрочный разбан
- при бане все темы пользователя удаляются
- база SQLite создаётся автоматически в data/app.sqlite

Как запустить локально
1. Установи PHP.
2. Открой папку проекта в терминале.
3. Выполни:
   php -S localhost:8000
4. Открой:
   http://localhost:8000

Можно также запустить через OpenServer / XAMPP / MAMP.

Важно
Просто двойным кликом по index.php этот проект не запустится.
Нужен PHP сервер.

Примечание
Игроки считаются отдельными командами. Ведущий даёт плюс или минус той команде, которая первой нажала кнопку.
